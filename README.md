# Mini SaaS Task Management System

A production-ready mini-SaaS Task Management application with secure authentication and multi-user functionality. Built with Node.js, Express, Sequelize (SQLite), React, and Tailwind CSS.

## Features
- **Authentication**: JWT-based secure signup and login.
- **Task Management**: Create, read, update, and delete your personal tasks.
- **Responsive UI**: Built with React and Tailwind CSS for a modern, mobile-friendly experience.

## Getting Started

### Backend Setup
```bash
cd backend
npm install
node server.js
```
*Note: The backend uses SQLite, so no external database configuration is required.*

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

Visit `http://localhost:5173` to view the application.
